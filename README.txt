Git et GitHub - Partie2 Activité

Cet exercice a pour objectif de mettre en pratique les notions acquises dans la partie 1 et 2.
Objectif :
1) Créer un repository git--> Pour cette exercice, il s'appellera projet_git.
2) Dans projet_git, je crée un fichier README.txt (ou j'explique le déroulement du projet).
3) Toujours dans projet_git, création d'au moins 2 autres fichiers, ici je choisis de créer trois autres fichiers avec des extensions différentes :
	coder1.md
	coder1.txt
	coder.py
4) Création d'au moins 4 commits dsitincts montrant l'ajout puis la modification de certains fichiers :
	commit1 coder1.md
	commit2 coder1.txt
	commit3 coder.py
	commit4 Ajout script dans coder.py
	commit5 README.txt
	commit6 Ajout commentaire dans coder1.txt
	commit7 Ajout commentaire dans coder1.md
5) Envoi du dossier projet_git (le repository git) sur GitHub
6) Indiquez l'Url de mon repository dans un document texte, qui sera ensuite compressé pour être envoyé.
